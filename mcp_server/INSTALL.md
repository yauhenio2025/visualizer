# Visualizer MCP Server - Quick Install

AI-powered document analysis with 47+ engines. Generates 4K visual diagrams or text reports from PDFs and documents.

## Setup (2 minutes)

### Step 1: Unzip to any folder

```bash
unzip visualizer-mcp-server.zip -d ~/visualizer-mcp-server
cd ~/visualizer-mcp-server
```

### Step 2: Ask Claude Code to set it up

Open Claude Code and say:

> "I unzipped the visualizer MCP server to ~/visualizer-mcp-server/. Please set it up for me.
> My Anthropic API key is sk-ant-... and my Gemini API key is AIza..."

Claude Code will:
- Create the virtual environment
- Install dependencies
- Create `.env.mcp` with your API keys
- Update your Claude Code config

### Step 3: Restart Claude Code

Quit and reopen Claude Code to load the new MCP server.

### Step 4: Test it

Say: "List the available visualizer engines"

## Manual Setup (if Claude Code setup doesn't work)

```bash
cd ~/visualizer-mcp-server

# 1. Create virtual environment
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 2. Create config file
cp .env.mcp.template .env.mcp
# Edit .env.mcp and add your API keys

# 3. Make scripts executable
chmod +x run-mcp-server.sh job_poller.py

# 4. Register with Claude Code
claude mcp add visualizer --transport stdio -- "$(pwd)/run-mcp-server.sh"

# 5. Restart Claude Code
```

## Troubleshooting

**MCP server not found:**
```bash
claude mcp list  # Should show "visualizer"
```

**No sound/notification:**
- Linux: Requires `notify-send` and `paplay` (usually pre-installed)
- macOS: Uses native notifications (works out of box)
- Windows: Uses PowerShell toast notifications

**API errors:**
- Check that your API keys are correct in `.env.mcp`
- Ensure keys have no extra spaces or newlines
